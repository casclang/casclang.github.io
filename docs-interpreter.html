The Casclang interpreter

What is an interpreter? To put it simply, it is a program which will take the code that you type, and then allow for it to be built. In this case, the Casclang interpreter is a terminal window, where you are able to input information by typing directly into it, and providing it matches the Casclang syntax, will then allow for it to be built.

• You can also paste your Casclang code from a file into the interpreter to then have it build, if you do not wish to type inside of a terminal.

The Casclang interpreter can be ran by double pressing the casclang-interpreter.exe file. If you are on macOS or Linux please install Mono and then launch the interpreter with mono casclang-interpreter.exe inside of a terminal.
