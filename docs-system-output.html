System.Output method

Definition

Namespace: System

Writes the text representation of the specified value or values to the standard output stream.

Methods & Submethods

Method Usage
Output Outputs information on the same line.
Output.line Outputs information to a new line.

Examples

// A simple "Hello, world!" Program:

Namespace {

System

}

System.Method.call {

Main()

}

System.Method.void {

Main()

{

System.Output {

"Hello, world"

}

}

Using the "Hello, world" example from our homepage, you can see a program that will display "Hello, world" to the console. Pretty basic. If we were to add another System.Output {} with information, it will append to the same line.

// Outputting with multiple lines:

Namespace {

System

}

System.Method.call {

Main()

}

System.Method.void {

Main()

{

System.Output.line {

"Hello, world"

}

System.Output.line {

"I am a program!"

}

}

Here we can see that the program is now outputting two lines. "Hello, world", then a new line, and then "I am a program!". If we were to use System.Output rather than System.Output.line, than these would both be on the same line.
