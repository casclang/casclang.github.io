Namespaces

Namespaces are heavily used in Casclang programming. .NET utilises namespaces as a way to organise its many methods and classes.

We need to add namespaces into our document to ensure that they can be utilised. You cannot write "System.Output {}" without first including the System namespace in your codebase.

Namespace {

System

}

To include a namespace in your code, you can use the "Namespace" method as shown above. From here you can use all methods that include the "System" namespace within your code.

You can add multiple namespace within the "Namespace" method, such as:

Namespace {

System

System.Collections.Generic

}
