System.Variable Method

Definition

Namespace: System

A declaration statement declares a new variable. Variables in Cacsclang are implicitly immutable and remain as such. As such variables must be modified utilising the "System.Variable.modify" submethod. Variables are strongly typed in Casclang and as such there is no implicit variable declaration.

Methods & Submethods

Method Usage
Variable.string Declares a variable as a string.
Variable.integer Declares a variable as an integer.
Variable.float Declares a variable as a floating point number.
Variable.boolean Declares a variable as a boolean.
Variable.modify Temporarly allows for a variable to be mutible.

Examples

// Creating and modifying variables:

Namespace {

System

}

System.Method.call {

Main()

}

System.Method.void {

Main()

{

System.Variable.integer {

a = 3

}

System.Variable.modify {

a = a + 6

}

}

Here we create an integer variable, called "a" and assign it a value of 3. Utilising the Variable.modify submethod we modify the variable and write "a = a + 6", which takes the value of a, which is 3, and add 6 to it. If we were to output "a" utilising System it would return a value of 9.

You can also utilise operators and operands with the Variable.modify submethod, such as "a++"" to increment the value of "a" by one.
