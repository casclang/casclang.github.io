Building your program

Once you have completed your program you may want to build it. This can be done with the Build{} method. Once this is done it will ask for a directory to build to, once provided a Program.cs file will be created.

You may wish to compile this Program.cs file into an executable binary. This can be done utilising Mono. First Mono will need to be installed from here, from there you can run csc Program.cs in your build directory to get the compiled executable.
