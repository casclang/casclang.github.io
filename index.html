Why Casclang?

Productivity

Casclang provides high level abstraction and changes the way you look at code by providing a new viewpoint, allowing you to look at programming in a Cascading style.

Blazingly fast

Casclang compiles into C# code blazingly fast, with most programs taking only a few seconds to fully compile.

Open Source

Casclang is free and open-source software. It has a great community that contributes to the source code and helps to develop its tooling.

What can you build?

Command Line Programs

By using the large System namespace, or any other namespace, you are able to build robust Command Line programs that work blazingly fast.

Libraries

Write Libraries for other programming languages, or even Casclang by utilising the Casclang language.
